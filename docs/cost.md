# 金額の意味 / What the Cost Means

[← README に戻る](../README.md)

> **⚠️ 表示される金額はあくまで概算値です。** transcript のトークン数 × 公開単価表からローカルで計算した参考値であり、Anthropic の請求額と一致することを保証するものではありません。「使いすぎに気づく」ための目安としてご利用ください。

`costLabel` は **表示ラベルを変えるだけ** で、計算方法そのものは変わりません。常にトークン数 × 単価で計算した同じ金額を使います。

- **`api_equivalent`(既定)**: Claude Pro / Max などの**定額プラン**を使っている場合、これは実際の請求額ではなく「もし従量課金の API で同じやり取りをしたらいくらか」の**参考換算値**です。通知の先頭に「API換算」と表示されます
- **`actual`**: **API キー**で従量課金利用している場合、この金額はほぼ実費に一致します。この場合は「API換算」ラベルを外して表示できます(`--label actual` または `config.json` で切替)

単価は Anthropic / OpenAI の公開レートを内蔵した単価表をベースに、[LiteLLM](https://github.com/BerriAI/litellm) が公開している価格データをキャッシュします。単価の取得は通常の`init`、`doctor`、`sweep`で行います。応答完了ごとのStop hookは、Claude Code / Codexを単価取得で待たせないよう、キャッシュと内蔵表だけを使います。プロンプトキャッシュも「5分保持」か「1時間保持」かで単価が異なる点まで区別して計算しています。

Claude Sonnet 5の内蔵単価は公式の期間限定価格に合わせ、2026年8月31日までは入力`$2/MTok`・出力`$10/MTok`、2026年9月1日以降は入力`$3/MTok`・出力`$15/MTok`を使います。

## 概算値である理由 / Why the numbers are estimates

`actual` 設定であっても、表示額は**請求書の代わりにはなりません**。あくまで概算値・目安として見てください。ズレが生じうる主な理由:

- **計算がローカルの近似**: Claude Code が transcript に記録した usage(トークン数)× 単価表で計算しており、Anthropic 側の請求計算そのものではありません
- **単価表の鮮度**: 単価は LiteLLM の公開データ + 内蔵表に依存します。新モデルの追加や価格改定への反映が遅れることがあります。新版モデルを名前の似た古いモデルの単価で推測せず、完全に一致する単価がなければ0円扱いの unknown model として通知・`doctor`に表示します
- **doctorのCodex 最新rollout合計**: 親/子を分類しない最新の単一rolloutを既存単価表で計算したAPI換算の参考値です。Claude Codeの直近額、別rollout、Codexサブエージェント料金とは合算せず、実請求額を示すものではありません
- **為替は変動**: 通常の¥表示は、有効期限内キャッシュ → オンライン取得 → 期限切れキャッシュ → `fx.fallbackRate`の順で選んだレートで換算します。`fx.fallbackRate`は通常のレートを固定する設定ではありません。`sweep`で取り込んだ過去分も**実行時点で選ばれたレート**で換算します(優先順位の詳細は[設定](configuration.md#為替レートとフォールバック--exchange-rate-fallback))
- **月予算の¥併記は例外**: USDで設定した予算額の¥表示だけは、オンラインレートではなく常に`fx.fallbackRate`を使います。使用率はUSD同士で計算するため、この表示用換算で使用率は変わりません(詳細は[月予算](monthly-budget.md))
- **sweepで過去額も変わり得る**: `sweep`は既存のコスト履歴を捨て、残っている元JSONLの対象レコードを実行時点の単価表・為替で再計算します。単価改定や為替変動があれば、同じ利用履歴でも以前の表示額とは一致しません
- **取りこぼしの可能性**: hook が発火しなかったターン(強制終了など)は記録されません(`sweep` で後から回収できます)

正確な請求額は [Anthropic Console](https://console.anthropic.com/)(API 利用)や各プランの請求情報で確認してください。
