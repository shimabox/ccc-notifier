# 月予算 / Monthly budget

[← README に戻る](../README.md)

月に使える金額(USD)を決めておくと、ダッシュボードに**その月の使用額 / 予算・使用率(%)**をプログレスバーで表示できます(使用率が 70% を超えると黄色、100% を超えると赤で警告)。`report-all.html` の**全履歴版**では、コスト推移のグラフで過去の月(や日・週)を選ぶと、そのカレンダー月に連動して表示が切り替わります(「通算」時は今月)。毎ターン更新される `report.html` の**期間限定版**では、埋め込み対象外も含む全履歴から集計した当月の値に固定され、期間選択には連動しません。**予算の設定だけで、使用を止めたり通知したりはしません**(あくまで「使いすぎに気づく」ための可視化です)。

設定方法は2通り:

- **`init` の対話**で「月の予算(USD)」を聞かれます。既定は **$400**(空欄のままだと $400。`0` で無効)。
- **`budget` コマンド**でいつでも変更できます。

```bash
ccc-notifier budget            # 現在の予算と当月の使用率を表示
ccc-notifier budget 400        # 月予算を $400 に設定
ccc-notifier budget 0          # 予算を解除(未設定に戻す)
```

予算はUSDで設定します(コストがUSD基準のため)。ダッシュボードと`budget`コマンドでは、USDで設定した**予算額**を$と¥で併記し、この¥表示には常に`fx.fallbackRate`を使います。オンラインの実勢レートではなく固定値を使うのは、為替取得のたびに目標額の¥表示が変わらないようにするためです。

この例外は予算額の¥併記だけです。今月の使用額の¥表示は各ターンを記録したときのレート、使用率は`今月の使用額(USD) ÷ 月予算(USD)`で計算するため、`fx.fallbackRate`を変えても使用率は変わりません。通常の為替レートの選択順は[設定](configuration.md#為替レートとフォールバック--exchange-rate-fallback)を参照してください。

予算未設定(`0`)のときはカードを表示しません。設定は`config.json`の`monthlyBudgetUSD`に保存されます。
